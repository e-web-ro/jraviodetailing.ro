# Jravio Detailing Website

This is a premium auto detailing website built with Next.js, Framer Motion, and Lucide Icons.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** CSS Modules / Styled JSX (custom dark theme)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Gallery:** Yet Another React Lightbox

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the pages (Home, Services, Pricing, Gallery, Contact).
- `src/components`: Reusable components (Navbar, Footer).
- `src/app/globals.css`: Global styles and theme variables.

## Deployment

The project is ready to be deployed on Vercel or any Node.js hosting.
